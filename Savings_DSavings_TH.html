ข้อกำหนดและเงื่อนไขการเปิดบัญชีเงินฝากออมทรัพย์ จัดให้
(เงื่อนไขนี้ให้มีผลบังคับใช้ตั้งแต่วันที่ 24 ตุลาคม 2562 เป็นต้นไป)
1. กรณีเปิดบัญชีผ่านช่องทางสาขา
1.1 เฉพาะบุคคลธรรมดา (ไม่อนุญาตสำหรับการเปิดบัญชีร่วม, บัญชี “เพื่อ” และบัญชี “โดย”)
1.2 จำนวนเงินฝากในการเปิดบัญชีขั้นต่ำ 500 บาท
2. กรณีเปิดบัญชีผ่านช่องทางออนไลน์
2.1 เฉพาะบุคคลธรรมดา (ไม่อนุญาตสำหรับการเปิดบัญชีร่วม, บัญชี “เพื่อ” และบัญชี “โดย”) ที่มีอายุระหว่าง 1 5-70 ปี มีสัญชาติไทย และสามารถเปิดผ่านช่องทางออนไลน์ได้เพียงบัญชีเดียวเท่านั้น
2.2 เป็นบัญชีเงินฝากออมทรัพย์ ประเภทไม่มีสมุดคู่ฝาก และไม่มีจำนวนเงินขั้นต่ำในการเปิดบัญชี
2.3 เปิดบัญชีได้ทุกวัน ตั้งแต่เวลา 06.00 น. - 21.00 น. ผ่าน Krungsri Mobile App (KMA)
2.4 สามารถทำรายการหรือธุรกรรมต่างๆ เกี่ยวกับบัญชี ผ่านช่องทางต่างๆ ได้ดังต่อไปนี้
1) ฝากด้วยเงินสดผ่านเครื่องรับฝากเงินสดอัตโนมัติ (Cash Deposit Machine – CDM) หรือโอนเงินจากบัญชีเงินฝากบัญชีอื่น
2) ถอนเงินด้วยบริการกดเงินไม่ใช้บัตร (Cardless ATM) ผ่านเครื่อง ATM ของธนาคาร หรือช่องทางอื่นๆ ตามที่ธนาคารกำหนด
3) โอนเงิน ชำระค่าสินค้าและบริการผ่าน Krungsri Online (KOL) และ Krungsri Mobile App (KMA)
2.5 สามารถผูกบัญชีเพื่อใช้บริการหักบัญชีอัตโนมัติผ่าน Krungsri Online (KOL) และ Krungsri Mobile App (KMA)
2.6 ไม่สามารถนำไปใช้เป็นหลักประกันการชำระหนี้ หรือหลักประกันใดๆ กับบุคคลอื่นได้
3. ธนาคารจะคำนวณดอกเบี้ยตามจำนวนเงินฝากคงเหลือ ณ สิ้นวัน และจะจ่ายดอกเบี้ยทุกเดือน ในวันก่อนวันทำการสุดท้ายของแต่ละเดือน หลังหักภาษี ณ ที่จ่าย (ถ้ามี)
4. การคำนวณดอกเบี้ยภาษีรับเป็นไปตามเงื่อนไขที่กรมสรรพากรกำหนด
5. ขอสงวนสิทธิ์ในการเปลี่ยนแปลงเงื่อนไขและวิธีการให้บริการโดยไม่ต้องแจ้งให้ทราบล่วงหน้า

ข้อกำหนดและเงื่อนไขการเปิดบัญชีเงินฝากผ่าน Krungsri Mobile Application
(เงื่อนไขนี้ให้มีผลบังคับใช้ตั้งแต่วันที่ 24 ตุลาคม 2562 เป็นต้นไป)
1. คุณสมบัติของผู้เปิดบัญชี เป็นบุคคลธรรมดาที่มีอายุตั้งแต่ 15 ปี แต่ไม่เกิน 70 ปี
2. ผู้เปิดบัญชีตกลงและรับทราบว่า การเปิดบัญชีเงินฝากผ่าน Krungsri Mobile Application ถือเป็นส่วนหนึ่งของการใช้บริการกรุงศรีออนไลน์ ซึ่งเป็นการเปิดบัญชีแบบไม่มีสมุดคู่ฝากและไม่มีการจัดทำเอกสารหลักฐานที่ต้องลงลายมือชื่อเป็นลายลักษณ์อักษร โดยผู้เปิดบัญชีตกลงผูกพันตามข้อตกลงการใช้ Krungsri Mobile Application, บริการกรุงศรีออนไลน์ และเงื่อนไขที่เกี่ยวข้องตามที่ธนาคารกำหนดทุกประการ
3. ในกรณีที่ผู้เปิดบัญชียังไม่เคยใช้ Krungsri Mobile Application ผู้เปิดบัญชีจะต้องดาวน์โหลด Krungsri Mobile Application เพื่อติดตั้งบนโทรศัพท์เคลื่อนที่ หรืออุปกรณ์อิเล็กทรอนิกส์อื่นใดตามที่ธนาคารกำหนด ในเครือข่ายที่ธนาคารรองรับให้สามารถเชื่อมต่อกับบริการนี้ได้ผ่านระบบอินเทอร์เน็ต โดยต้องทำการดาวน์โหลดผ่านช่องทางและดำเนินการตามที่ธนาคารกำหนด เพื่อใช้เป็นช่องทางในการเปิดบัญชีเงินฝาก และผู้เปิดบัญชีจะต้องสมัครใช้บริการ Krungsri Mobile Application ในคราวเดียวกับการเปิดบัญชีดังกล่าวด้วย
4. ผู้เปิดบัญชีตกลงและรับทราบว่า รูปถ่าย ข้อมูลทางชีวมิติ (Biometric) (เช่น ภาพจำลองใบหน้า หรือลายนิ้วมือ เป็นต้น) เอกสารและข้อมูลอื่นใดที่ธนาคารได้รับจากผู้เปิดบัญชีในการเปิดบัญชีเงินฝากผ่าน Krungsri Mobile Application ไม่ว่าผ่านช่องทางใดๆ ซึ่งรวมถึงช่องทางอิเล็กทรอนิกส์ เป็นข้อมูลที่ระบุและพิสูจน์ตัวตนผู้ขอใช้บริการ รวมทั้งเป็นข้อมูลที่ถูกต้องครบถ้วน เป็นปัจจุบัน และเป็นจริงทุกประการ โดยธนาคารสามารถนำมาใช้ในการให้บริการนี้ได้โดยชอบด้วยกฎหมาย ในกรณีที่เอกสารหรือข้อมูลดังกล่าวไม่ครบถ้วน ไม่เป็นปัจจุบัน หรือไม่ถูกต้องตรงกับความเป็นจริง ผู้เปิดบัญชีตกลงยินยอมให้ธนาคารมีสิทธิระงับหรือปิดบัญชีเงินฝากที่เปิดผ่าน Krungsri Mobile Application เมื่อใดก็ได้โดยไม่ต้องแจ้งให้ทราบล่วงหน้า โดยธนาคารไม่ต้องรับผิดชอบต่อผู้เปิดบัญชีหรือบุคคลอื่นใดในความเสียหายที่เกิดขึ้น
5. การเปิดบัญชีเงินฝากผ่าน Krungsri Mobile Application
5.1 ผู้เปิดบัญชีสามารถเปิดบัญชีเงินฝากได้ตามประเภทและจำนวนบัญชีตามที่ธนาคารกำหนด โดยผู้เปิดบัญชีตกลงและยอมรับว่า เงื่อนไขการเปิดบัญชีเงินฝากผ่าน Krungsri Mobile Application เป็นส่วนหนึ่งของเงื่อนไขการเปิดบัญชีเงินฝากของธนาคารด้วย
5.2 การเปิดบัญชีเงินฝากจะสมบูรณ์ ต่อเมื่อผู้เปิดบัญชีได้ดำเนินการตามขั้นตอนที่ธนาคารกำหนดโดยครบถ้วนสมบูรณ์ และได้รับข้อความแจ้งยืนยันการเปิดบัญชีเงินฝากเรียบร้อยแล้ว
5.3 ผู้เปิดบัญชีสามารถตรวจสอบรายการเดินบัญชี ซึ่งรวมถึงการฝาก ถอน โอน ชำระค่าสินค้า/บริการ ยอดเงินคงเหลือ และข้อมูลอื่นใดเกี่ยวกับบัญชีเงินฝาก ผ่านบริการ Krungsri Mobile Application หรือช่องทางอื่นใดตามที่ธนาคารกำหนด โดยผู้เปิดบัญชีตกลงและยอมรับว่ารายละเอียดและข้อมูลที่ปรากฏในระบบของธนาคารเป็นข้อมูลที่ถูกต้องและผูกพันผู้เปิดบัญชีทุกประการ
6. ชื่อบัญชีที่เปิดผ่าน Krungsri Mobile Application จะใช้ชื่อเดียวกับชื่อผู้เปิดบัญชี โดยไม่สามารถใช้ชื่ออื่นหรือนามแฝง หรือเปิดเป็นบัญชีร่วมได้ โดยผู้เปิดบัญชีต้องเปิดบัญชีเงินฝากด้วยตัวเองเท่านั้น ไม่สามารถให้ผู้อื่นเปิดแทนได้ ในกรณีที่ผู้เปิดบัญชียินยอมให้บุคคลอื่นที่มีใบหน้าเหมือนหรือคล้ายกับผู้เปิดบัญชี หรือเป็นฝาแฝดของผู้เปิดบัญชี เป็นผู้เปิดบัญชีแทน ผู้เปิดบัญชีตกลงให้ธนาคารมีสิทธิระงับการเปิดบัญชี ปิดบัญชี หรือระงับการให้บริการนี้ได้ทันที และหากมีความเสียหายใดๆ เกิดขึ้น ผู้เปิดบัญชีตกลงรับผิดชอบในความเสียหายที่เกิดขึ้นทั้งสิ้น
7. ผู้เปิดบัญชีตกลงและรับทราบว่า หากผู้เปิดบัญชีมีฝาแฝดที่เกิดจากการปฏิสนธิจากไข่ใบเดียวกัน (Identical Twins) ซึ่งมีเพศเดียวกัน และมีใบหน้าเหมือนหรือคล้ายกันแล้ว ธนาคารจะไม่ยินยอมให้เปิดบัญชีเงินฝากโดยเด็ดขาด ซึ่งธนาคารถือเป็นเงื่อนไขที่สำคัญสำหรับการเปิดบัญชีเงินฝากผ่าน Krungsri Mobile Application ในกรณีที่ธนาคารตรวจสอบพบในภายหลังว่าผู้เปิดบัญชีมีฝาแฝดในลักษณะดังกล่าว ผู้เปิดบัญชีตกลงและยินยอมให้ธนาคารมีสิทธิปิดบัญชี หรือระงับการให้บริการนี้ได้ทันที โดยไม่ต้องแจ้งให้ทราบล่วงหน้าก่อนแต่อย่างใด และหากมีความเสียหายใดๆ เกิดขึ้น ผู้เปิดบัญชีตกลงเป็นผู้รับผิดชอบในความเสียหายใดๆ ที่เกิดขึ้นแต่เพียงฝ่ายเดียว
8. จำนวนเงินฝากขั้นต่ำสำหรับการเปิดบัญชีผ่าน Krungsri Mobile Application เป็นไปตามที่ธนาคารกำหนดไว้สำหรับการเปิดบัญชีเงินฝากแต่ละประเภท โดยผู้เปิดบัญชีตกลงฝากเงินเข้าบัญชีตามขั้นตอนและวิธีการเปิดบัญชีที่ธนาคารกำหนด
9. เนื่องจากการเปิดบัญชีผ่าน Krungsri Mobile Application เป็นบริการเปิดบัญชีแบบไม่มีสมุดคู่ฝาก ซึ่งอาจมีข้อจำกัดในการทำรายการหรือธุรกรรมเกี่ยวกับบัญชีเงินฝาก เช่น การถอนหรือโอนเงินออกจากบัญชีจะกระทำได้เฉพาะผ่านช่องทางและวิธีการที่ธนาคารกำหนดเท่านั้น ดังนั้น ผู้เปิดบัญชีตกลงจะไม่เรียกร้องให้ธนาคารต้องรับผิดชอบอันเนื่องมาจากข้อจำกัดในการทำรายการหรือธุรกรรมดังกล่าว
10. ผู้เปิดบัญชีรับทราบว่า ธนาคารจะทำการบันทึกข้อมูลต่างๆ ที่ทำผ่านบัญชีเงินฝากที่เปิดผ่าน Krungsri Mobile Application เช่น การฝาก ถอน โอน หรือยอดเงินคงเหลือ และข้อมูลใดๆ เกี่ยวกับบัญชีเงินฝากไว้ในฐานข้อมูลของธนาคาร ซึ่งผู้เปิดบัญชีสามารถตรวจสอบรายละเอียดและข้อมูลต่างๆ ได้ผ่าน Krungsri Mobile Application หรือช่องทางอื่นใดตามที่ธนาคารกำหนด โดยผู้เปิดบัญชียอมรับว่า รายละเอียดและข้อมูลต่างๆ ที่ปรากฎอยู่ในฐานข้อมูลของธนาคารเป็นข้อมูลที่ความถูกต้องและครบถ้วน
11. ในการทำรายการหรือธุรกรรมต่างๆ ผ่านบัญชีเงินฝากที่เปิดผ่าน Krungsri Mobile Application ผู้เปิดบัญชีตกลงให้ถือว่า รายการหรือธุรกรรมต่างๆ ที่ได้ทำตามเงื่อนไขของแต่ละช่องทาง เป็นรายการหรือธุรกรรมที่ถูกต้องและผูกพันผู้เปิดบัญชีโดยไม่ต้องทำเอกสารหรือหลักฐานเพิ่มเติมแต่ละคราวอีก เว้นแต่ธนาคารจะกำหนดไว้เป็นอย่างอื่น และเมื่อผู้เปิดบัญชีทำรายการหรือธุรกรรมต่างๆ ครบถ้วนสมบูรณ์ตามเงื่อนไขที่ธนาคารกำหนดแล้ว ผู้เปิดบัญชีตกลงให้ถือว่ายอดเงินคงเหลือในฐานข้อมูลของธนาคารเป็นยอดเงินคงเหลือที่ถูกต้อง
12. ผู้เปิดบัญชีตกลงยินยอมให้ธนาคารมีสิทธิหักเงินในบัญชีเงินฝากที่เปิดผ่าน Krungsri Mobile Application ที่ผู้เปิดบัญชีมีอยู่กับธนาคาร เพื่อชำระหนี้ทั้งหมดหรือแต่บางส่วนที่เกิดขึ้นระหว่างผู้เปิดบัญชีกับธนาคารได้โดยไม่จำต้องแจ้งให้ผู้เปิดบัญชีทราบล่วงหน้า และไม่ต้องรอให้รายการฝากเงินของบัญชีเงินฝากดังกล่าวถึงกำหนด (หากมี)
13. ผู้เปิดบัญชีตกลงและรับทราบว่า ในกรณีที่ Krungsri Mobile Application ไม่สามารถใช้งานได้ ไม่ว่าด้วยเหตุใด ผู้เปิดบัญชีตกลงไม่ถือเอาเหตุดังกล่าวมาเรียกร้องให้ธนาคารต้องรับผิดชอบ
14. การนำบัญชีเงินฝากที่เปิดผ่าน Krungsri Mobile Application ไปใช้เป็นหลักประกันการชําระหนี้หรือหลักประกันใดๆ กับบุคคลอื่นใดได้หรือไม่นั้น ให้เป็นไปตามหลักเกณฑ์และเงื่อนไขที่ธนาคารกำหนด
15. ผู้เปิดบัญชีจะโอนสิทธิในบัญชีเงินฝาก หรือสิทธิในการรับเงินฝากไม่ว่าทั้งหมดหรือแต่บางส่วนให้แก่บุคคลอื่นใดไม่ได้ทั้งสิ้น เว้นแต่จะได้รับความยินยอมจากธนาคารก่อน

17. บรรดาเอกสาร หนังสือ หรือการแจ้งข้อมูลข่าวสารใดๆ ซึ่งธนาคารได้ส่งหรือแจ้งไปยังที่อยู่ สถานที่ทำงาน หมายเลขโทรศัพท์ หมายเลขโทรสาร หรือจดหมายอิเล็กทรอนิกส์ (E-mail) ของผู้เปิดบัญชีตามที่แจ้งไว้กับธนาคาร ผู้เปิดบัญชียินยอมให้ถือว่าธนาคารได้ส่งหรือแจ้งแก่ผู้เปิดบัญชีโดยชอบแล้ว ในกรณีที่ผู้เปิดบัญชีเปลี่ยนแปลงที่อยู่หรือสถานที่ติดต่อดังกล่าวข้างต้น ผู้เปิดบัญชีตกลงแจ้งให้ธนาคารทราบเป็นลายลักษณ์อักษรหรือตามวิธีการที่ธนาคารกำหนด
18. ผู้เปิดบัญชีตกลงและยอมรับว่า วิธีการดําเนินการใดๆ โดยผ่านอุปกรณ์อิเล็กทรอนิกส์ และ/หรือระบบอิเล็กทรอนิกส์ เป็นวิธีการที่น่าเชื่อถือและยอมรับได้ระหว่างผู้เปิดบัญชีและธนาคาร รวมทั้งรับรองว่า เอกสารและข้อมูลอื่นใดที่ธนาคารเก็บรวบรวมซึ่งได้รับจากผู้เปิดบัญชีด้วยวิธีการดังกล่าว หรือที่ใช้ประกอบการดําเนินการดังกล่าวนั้น เป็นเอกสารและข้อมูลที่เชื่อถือได้และถูกต้องแท้จริงเสมือนเอกสารที่เป็นลายลักษณ์อักษรทุกประการ และใช้บังคับได้ระหว่างผู้เปิดบัญชีและธนาคาร
19. ผู้เปิดบัญชีตกลงและยอมรับว่า การเปิดบัญชีเงินฝากและการลงนามตามข้อกำหนดและเงื่อนไขนี้ เป็นการกระทําผ่านอุปกรณ์อิเล็กทรอนิกส์ ซึ่งหากได้กระทําไปโดยมีการยืนยันด้วยสิ่งที่ผู้เปิดบัญชีใช้ในกระบวนการพิสูจน์และยืนยันตัวตน เช่น User Name / Password / PIN / One Time Password (OTP) / ข้อมูลทางชีวมิติ (Biometric) (เช่น ภาพจำลองใบหน้า หรือลายนิ้วมือ เป็นต้น) หรือรหัสอื่นใด ให้ถือว่าถูกต้องสมบูรณ์นับตั้งแต่เวลาที่มีการยืนยันการทำธุรกรรมเปิดบัญชีเงินฝาก และให้มีผลผูกพันผู้เปิดบัญชี รวมทั้งให้ถือว่าเป็นการที่ผู้เปิดบัญชีได้ลงลายมือชื่ออิเล็กทรอนิกส์ให้ไว้แก่ธนาคารเพื่อเป็นหลักฐาน โดยผู้เปิดบัญชีไม่จําเป็นต้องลงลายมือชื่อในเอกสารใดๆ ทั้งสิ้น และตกลงให้ธนาคารสามารถใช้ข้อมูลการเปิดบัญชีเงินฝากและข้อกำหนดและเงื่อนไขนี้ เป็นต้นฉบับเอกสารเพื่อใช้ในการพิสูจน์ว่าผู้เปิดบัญชีได้เปิดบัญชีเงินฝาก และใช้ในการดำเนินการทางกฎหมายได้ทุกประการ
20. หากมีความเสียหายใดๆ เกิดขึ้นจากเครื่องมือหรืออุปกรณ์การใช้บริการของผู้เปิดบัญชี หรือเกิดจากระบบเครือข่ายโทรศัพท์มือถือ ธนาคารไม่ต้องรับผิดชอบต่อผู้เปิดบัญชี หรือผู้รับโอนเงิน หรือบุคคลอื่นใดทั้งสิ้น
21. ในกรณีที่ธนาคารมีความจำเป็นต้องเปิดเผยข้อมูลของผู้เปิดบัญชี และ/หรือข้อมูลทางการเงิน ธุรกรรม หรือข้อมูลใดๆ ที่เกี่ยวข้องกับบัญชีเงินฝากให้แก่บุคคลหรือหน่วยงานใด ภายใต้บทบัญญัติของกฎหมาย คำสั่ง หรือกฎระเบียบของหน่วยงานที่มีอำนาจกำกับดูแลธนาคาร ผู้รับชำระเงิน ผู้รับโอนเงิน บุคคลหรือหน่วยงานใดๆ เพื่อประโยชน์ในการดำเนินการหรือให้บริการตามข้อกำหนดและเงื่อนไขนี้ ผู้เปิดบัญชีตกลงและให้สิทธิแก่ธนาคารในการเปิดเผยข้อมูล รายงานข้อมูล จัดทำรายงาน หรือข้อมูลใดๆ ดังกล่าวข้างต้นของผู้เปิดบัญชี ให้แก่บุคคลหรือหน่วยงานนั้นได้ทุกประการ และให้สิทธิแก่ธนาคารในการดำเนินการตามข้อนี้มีอยู่ตลอดไปถึงแม้จะได้มีการปิดบัญชีเงินฝากไปแล้ว
22. ข้อตกลงการใช้บริการนี้ อยู่ภายใต้บังคับและเป็นส่วนหนึ่งของข้อตกลงและเงื่อนไขการใช้บริการบัญชีเงินฝากทุกประเภทของธนาคาร รวมทั้งข้อตกลงและเงื่อนไขอื่นที่เกี่ยวข้องกับการเปิดบัญชีเงินฝาก โดยผู้เปิดบัญชีตกลงยินยอมผูกพันตามข้อตกลงและเงื่อนไขดังกล่าวทุกประการ
23. ผู้เปิดบัญชีรับทราบจำนวนเงินฝากขั้นตํ่าเพื่อทำการเปิดบัญชีเงินฝากครั้งแรก (ถ้ามี) จำนวนเงินขั้นตํ่าสำหรับการนำฝากในแต่ละครั้ง ตลอดจนหน้าที่ในการรักษายอดเงินคงเหลือในบัญชีเงินฝากตามประกาศของธนาคารที่ใช้บังคับอยู่ในขณะนี้ ในกรณีที่ผู้เปิดบัญชีไม่ได้ทำการเคลื่อนไหวทางบัญชีและมียอดเงินคงเหลือในบัญชีเงินฝากตํ่ากว่าจำนวนที่ธนาคารกำหนด ผู้เปิดบัญชียินยอมให้ธนาคารเรียกเก็บค่าธรรมเนียมในการรักษาบัญชีเงินฝาก ตามหลักเกณฑ์ที่ธนาคารประกาศกำหนด และหากธนาคารได้มีประกาศเปลี่ยนแปลงเงื่อนไขเกี่ยวกับการรักษายอดเงินคงเหลือในบัญชี หรืออัตราค่าธรรมเนียมในการรักษาบัญชีเงินฝาก ผู้เปิดบัญชีตกลงยินยอมผูกพันตามเงื่อนไขของประกาศดังกล่าวนั้นทุกประการ โดยที่ธนาคารไม่จำต้องแจ้งให้ผู้เปิดบัญชีทราบแต่อย่างใด ทั้งนี้ หากบัญชีเงินฝากของผู้เปิดบัญชีมียอดเงินคงเหลือในบัญชีเงินฝากเท่ากับศูนย์ และเมื่อธนาคารเห็นสมควร ผู้เปิดบัญชียินยอมให้ธนาคารปิดบัญชีเงินฝากของผู้เปิดบัญชีได้โดยไม่จำต้องแจ้งให้ผู้เปิดบัญชีทราบแต่อย่างใด
24. ผู้เปิดบัญชีจะได้รับดอกเบี้ยเงินฝากในอัตราที่ธนาคารประกาศกำหนดไว้ในข้อกำหนดและเงื่อนไขนี้ และ/หรือข้อกำหนดและเงื่อนไขสำหรับการเปิดบัญชีเงินฝากแต่ละประเภท รวมทั้งข้อกำหนดและเงื่อนไขที่ธนาคารจะประกาศเพิ่มต่อไปในอนาคต พร้อมทั้งยินยอมให้ธนาคารเปลี่ยนแปลงอัตราดอกเบี้ยเงินฝากและเงื่อนไขการจ่ายดอกเบี้ยได้ โดยธนาคารจะแจ้งให้ผู้เปิดบัญชีทราบภายใต้หลักเกณฑ์ตามประกาศ หรือระเบียบของธนาคารแห่งประเทศไทย หรือหน่วยงานราชการอื่นที่เกี่ยวข้องกับเรื่องดังกล่าว และผู้เปิดบัญชียอมรับตามอัตราและเงื่อนไขการจ่ายดอกเบี้ยเงินฝากที่ธนาคารได้กำหนดนั้นทุกประการ
25. ในกรณีที่ผู้เปิดบัญชียกเลิกการใช้บริการ Krungsri Mobile Application หรือไม่สามารถเข้าใช้บริการ Krungsri Mobile Application ได้ บัญชีเงินฝากจะยังคงสถานะการเปิดบัญชีอยู่ แต่ผู้เปิดบัญชีจะไม่สามารถถอนเงินหรือโอนเงินผ่านบัญชีได้ โดยผู้เปิดบัญชีสามารถทำการปิดบัญชีได้ที่สาขาสำนักพระรามที่ 3 ในวันและเวลาทำการของธนาคาร ทั้งนี้ ธนาคารมีสิทธิที่จะปิดบัญชีเงินฝากได้ตามดุลพินิจของธนาคาร โดยไม่จำเป็นต้องแจ้งให้ผู้เปิดบัญชีทราบล่วงหน้า
26. ผู้เปิดบัญชีรับทราบและตกลงว่า ธนาคารมีสิทธิที่จะระงับการฝาก ถอน โอน หรือธุรกรรมทางการเงินต่างๆ หรือจำกัดจำนวนเงินฝาก หรือคืนเงินฝากทั้งหมดหรือบางส่วน หรือปิดบัญชีเงินฝากของผู้เปิดบัญชีไม่ว่าเมื่อใดและไม่ว่าเหตุใดก็ตามแต่ดุลพินิจของธนาคาร โดยไม่จำเป็นต้องแจ้งให้ผู้เปิดบัญชีทราบล่วงหน้า โดยเฉพาะอย่างยิ่งเมื่อผู้เปิดบัญชีเป็นผู้มีหนี้สินล้นพ้นตัว หรือถูกพิทักษ์ทรัพย์ หรือเป็นบุคคลล้มละลาย หรือถูกอายัดทรัพย์สิน หรือมีเหตุอันควรสงสัยหรือเชื่อได้ว่าเอกสารแสดงตนหรือเอกสารประกอบอื่นๆ ที่ผู้เปิดบัญชียื่นไว้ต่อธนาคารในการเปิดบัญชีเงินฝากไม่ถูกต้อง หรือเป็นบุคคลที่ต้องสงสัยว่ามีส่วนเกี่ยวข้องในกิจกรรมที่ผิดกฎหมาย ระเบียบ ข้อบังคับ คำสั่ง หรือตามนโยบายใดๆ ของทางราชการหรือของธนาคาร หรือได้ประพฤติผิดข้อตกลงนี้ หรือบัญชีเงินฝากถูกนำไปใช้หรือมีพฤติการณ์น่าสงสัยว่ามีส่วนเกี่ยวข้องกับการกระทำความผิดตามกฎหมายว่าด้วยการป้องกันและปราบปรามการฟอกเงินหรือการสนับสนุนทางการเงินแก่การก่อการร้าย หรือไม่เป็นไปตามกฎหมายหรือกฎเกณฑ์ที่เกี่ยวข้อง หรือกระทำการหรืองดเว้นกระทำการใดซึ่งธนาคารเห็นว่าเป็นการใช้บัญชีเงินฝากโดยไม่สมควร ทั้งนี้ ผู้เปิดบัญชีจะไม่เรียกร้องค่าเสียหายใดๆ จากธนาคาร และธนาคารไม่ต้องรับผิดชอบในความเสียหายใดๆ ที่อาจเกิดขึ้นจากการดำเนินการดังกล่าวข้างต้นต่อผู้เปิดบัญชีแต่อย่างใด
27. ในกรณีที่ธนาคารใช้สิทธิปิดบัญชีเงินฝากของผู้เปิดบัญชี และยังคงมีเงินของผู้เปิดบัญชีคงเหลืออยู่ในบัญชีเงินฝาก ผู้เปิดบัญชีตกลงยินยอมให้เป็นดุลพินิจของธนาคารในการนำเงินฝากดังกล่าวไปวาง ณ สำนักงานวางทรัพย์ กรมบังคับคดี โดยถือว่าผู้เปิดบัญชีได้รับเงินฝากคืนจากธนาคารโดยถูกต้องครบถ้วน และธนาคารไม่ต้องคิดคำนวณดอกเบี้ยเงินฝากให้แก่ผู้เปิดบัญชีต่อไปอีก นับแต่วันที่ธนาคารได้นำเงินฝากไปวาง ณ สำนักงานวางทรัพย์ กรมบังคับคดี เป็นต้นไป
28. ผู้เปิดบัญชีทราบดีว่า ในกรณีที่รายการทางบัญชีมีการระบุจำนวนเงินที่ผู้เปิดบัญชี และ/หรือบุคคลอื่นใดได้นำฝาก ถอน โอน หรือด้วยวิธีอื่นใด และ/หรือระบุดอกเบี้ยที่ผู้เปิดบัญชีพึงได้รับไม่ถูกต้องตรงตามความเป็นจริงอันเกิดจากการสำคัญผิดหรือผิดพลาดไม่ว่าด้วยประการใดๆ ก็ตาม โดยผู้เปิดบัญชีไม่มีสิทธิในเงินจำนวนนั้น ผู้เปิดบัญชียินยอมให้ธนาคารหักเงินในบัญชีเงินฝากและปรับปรุงแก้ใขรายการทางบัญชีดังกล่าวให้ถูกต้องตรงตามความเป็นจริงได้ทันทีโดยไม่จำเป็นต้องแจ้งให้ผู้เปิดบัญชีทราบแต่อย่างใด ทั้งนี้ เมื่อใดก็ตามที่ผู้เปิดบัญชีทราบถึงข้อผิดพลาดในการทำรายการของธนาคารเกี่ยวกับบัญชีของผู้เปิดบัญชีตามที่กล่าวไว้ข้างต้นแล้ว แต่บัญชีเงินฝากของผู้เปิดบัญชีไม่มีเงินคงเหลือ หรือมีแต่ไม่เพียงพอให้ธนาคารหักเพื่อปรับปรุงหรือแก้ไขรายการผิดพลาดที่เกิดขึ้น ผู้เปิดบัญชียินยอมชดใช้เงินจำนวนดังกล่าวคืนจนครบถ้วน พร้อมดอกเบี้ยในอัตราสูงสุดที่ธนาคารใช้เรียกเก็บจากลูกค้าที่ปฏิบัติผิดเงื่อนไขตามหลักเกณฑ์ที่ธนาคารแห่งประเทศไทย หรือกฎหมายว่าด้วยการธนาคารพาณิชย์ หรือกฎหมายว่าด้วยธุรกิจสถาบันการเงินกำหนดนับแต่วันที่ผู้เปิดบัญชีได้รับเงินจำนวนดังกล่าวไป
29. ผู้เปิดบัญชีรับทราบและยอมรับว่า เมื่อธนาคารทราบว่าเจ้าของบัญชีเงินฝากถึงแก่กรรม ธนาคารจะระงับการจ่ายเงินฝากจากบัญชีเงินฝากดังกล่าวทันที และให้ผู้จัดการมรดกตามคำสั่งศาลของเจ้าของบัญชีเงินฝากที่ถึงแก่กรรมเท่านั้นที่มีสิทธิที่จะรับเงินฝากคืนจากธนาคาร โดยนำหลักฐานต่างๆ ที่ธนาคารต้องการมาพิสูจน์แสดงต่อธนาคารจนเป็นที่พอใจธนาคารจึงจะจ่ายเงินฝากคืนให้
30. ผู้เปิดบัญชีรับทราบและยินยอมชำระค่าธรรมเนียม ค่าบริการ และค่าใช้จ่ายใดๆ เกี่ยวกับการรับฝากเงินตามที่ธนาคารได้ประกาศกำหนด โดยผู้เปิดบัญชียินยอมให้ธนาคารหักเงินจากบัญชีเงินฝากของผู้เปิดบัญชีเพื่อชำระหนี้ค่าธรรมเนียม ค่าบริการ และค่าใช้จ่ายใดๆ ที่เกี่ยวกับบัญชีเงินฝากตามที่กล่าวมา รวมทั้งหนี้อื่นๆ ทุกลักษณะทุกประเภทหนี้ที่ผู้เปิดบัญชีค้างชำระอยู่กับธนาคารได้ทันทีโดยไม่จำต้องแจ้งให้ผู้เปิดบัญชีทราบก่อน ทั้งนี้ ผู้เปิดบัญชีขอสละสิทธิเรียกร้องในเงินฝากจำนวนที่ธนาคารจะหักนั้น ตั้งแต่เวลาที่หนี้ และ/หรือภาระผูกพันตามที่กล่าวข้างต้นนั้นได้เกิดขึ้นเป็นต้นไป
31. ผู้เปิดบัญชียินยอมผูกพันและตกลงปฏิบัติตามข้อตกลงและเงื่อนไขการฝากเงินนี้ ตลอดจนระเบียบและวิธีปฏิบัติอื่นๆ ของธนาคาร รวมทั้งอัตราและวิธีการเรียกเก็บค่าธรรมเนียม ค่าใช้จ่าย และ/หรือค่าบริการที่เกี่ยวกับการรับฝากเงินที่มีอยู่แล้ว และที่ธนาคารจะได้แก้ไขเปลี่ยนแปลงเพิ่มเติมหรือยกเลิกในภายหน้า โดยให้มีผลผูกพันผู้เปิดบัญชีโดยธนาคารไม่จำต้องแจ้งให้ผู้เปิดบัญชีทราบก่อน แต่หากการแก้ไข เปลี่ยนแปลง เพิ่มเติม และยกเลิกดังกล่าวทำให้ผู้เปิดบัญชีเสียประโยชน์ ธนาคารเพียงแต่แจ้งให้ผู้เปิดบัญชีทราบภายใต้หลักเกณฑ์ตามประกาศ กฎ หรือระเบียบของธนาคารแห่งประเทศไทย หรือหน่วยงานราชการอื่นที่เกี่ยวข้องกับเรื่องดังกล่าวเท่านั้น
32. ผู้เปิดบัญชีตกลงยินยอมผูกพันปฏิบัติตามเงื่อนไขการเปิดบัญชีเงินฝากผ่าน Krungsri Mobile Application รวมทั้งข้อกำหนดและเงื่อนไขอื่นๆ ที่ได้แสดงให้ปรากฏบนหน้าจอโทรศัพท์มือถือ หรืออุปกรณ์อิเล็กทรอนิกส์อื่นใดของผู้เปิดบัญชี ในขณะทำรายการเปิดบัญชีเงินฝาก ซึ่งผู้เปิดบัญชีได้อ่านและเข้าใจเป็นอย่างดีแล้ว โดยผู้เปิดบัญชีตกลงยอมรับว่าเป็นหน้าที่ของผู้เปิดบัญชีที่จะต้องทำการบันทึก (Save As) หรือสั่งพิมพ์ (Print) ข้อกำหนดและเงื่อนไขดังกล่าวเพื่อเก็บไว้เป็นหลักฐาน และผู้เปิดบัญชีตกลงว่า เมื่อผู้เปิดบัญชีได้เปิดบัญชีเงินฝากตามวิธีการที่ธนาคารกำหนดเรียบร้อยแล้ว ผู้เปิดบัญชีตกลงยินยอมผูกพันตามข้อตกลงและเงื่อนไขที่เกี่ยวข้องตามที่ธนาคารกำหนดทุกประการ และให้ถือว่าข้อมูลหรือหลักฐานเกี่ยวกับการเปิดบัญชีเงินฝาก เป็นส่วนหนึ่งของเงื่อนไขการเปิดบัญชีเงินฝากของธนาคารด้วย ทั้งนี้ หากมีความเสียหายใดๆ เกิดขึ้นอันเนื่องมาจากการที่ผู้เปิดบัญชีไม่ปฏิบัติตามเงื่อนไขดังกล่าว ผู้เปิดบัญชีตกลงเป็นผู้รับผิดชอบโดยจะไม่เรียกร้องให้ธนาคารต้องรับผิดชอบแต่อย่างใด
